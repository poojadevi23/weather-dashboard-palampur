# weather-dashboard-palampur
A real-time weather analytics dashboard built using Streamlit and WeatherAPI. This project provides live weather updates, key metrics, and forecast visualization for Palampur, Himachal Pradesh.

Features:
* Real-time temperature, humidity, and wind speed
* Current weather conditions
* Temperature forecast visualization
* Live API-based data (no static dataset)
* Deployed online and shareable

 Tech Stack:
* Python
* Streamlit
* Plotly
* WeatherAPI

 How It Works:
* Fetches real-time data from WeatherAPI
* Processes JSON data using Pandas
* Displays insights using Streamlit UI and Plotly charts
